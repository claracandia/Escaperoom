【VRChat向け】複数アバター対応 即応部隊"RAZE"

【更新履歴】
2022.06/27　v1.0


VRChatでの利用を想定した武器の3Dモデルです。
※画像中のアバター等は本商品に含まれません


＜使用方法＞
・VRChatへの導入
　Unity 2019.4.31f1上で
Assets → Import Package → Custom Package
を選択し、FULL ATTACK.unitypackageを読み込みます。
生成されたフォルダのうち、Teir 6 Performance　フォルダの直下にあるPrefabフォルダ内の各PrefabをSceneまたはHierarchyにドラッグ＆ドロップすることで配置することができます。
　アバターなど身につけさせたい位置へ各種Prefabを移動させ、Hierarchy上で対応するボーンに配置してください。

＜シェーダーについて＞

本製品の推奨シェーダーは【liltoon】となっております。

事前にliltoonをご使用のUnityProjectにインポートしてから使用して頂けると円滑にご利用いただけます。



＜内容物＞
RAZE COMPLITE.unitypackage(prefab、FBXモデル、テクスチャを含む)
RAZE WEAPON SP.unitypackage

【シェイプキー】

"HOUNDS M2022"

・HAMMER -ハンマーが可動します
・SLIDE LOCK -スライドロックが上がります
・SLIDE -スライドが後退します

"PitViper.300"

・handle -チャージングハンドルが後退します
・cover -ダストカバーが開きます
・back -ボルトが後退します
・semi -セレクターがセミオートになります
・full -セレクターがフルオートになります

"PitViper.300 and M243"

F sight(key1)-フリップアップサイトが可動します

R sight(Key1)-フリップアップサイトが可動します

"NVG"

・UP -ナイトビジョンが上に上がります
